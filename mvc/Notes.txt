-- Métodos --
index - quando carrega o controller e não passo o método

show - quando quero pegar um registro em especifico

edit - mostra formulário para atualizar registro

update - atualiza o registro do banco

create - mostrar o registro do banco

store - Cadastrar registro no banco

destroy - deletar registro do banco ou logout do sistema 

==== MVC ====
Models - Serve para trabalhar com o banco de dados
Views - Trabalha com os templates com o que o usuário vai ver na tela
Controllers - Vai servir para gerenciar tudo isso
